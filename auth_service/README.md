# auth_service

## 🚀 Setup Instructions

1. Create virtual environment 
   `python -m venv venv && source venv/bin/activate`

2. Install dependencies 
   `pip install -r requirements.txt`

3. Run migrations 
   `python manage.py migrate`

4. Start development server
   `python manage.py runserver`
